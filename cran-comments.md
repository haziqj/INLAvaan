## Release summary

This is a patch release. It makes INLAvaan compatible with the upcoming lavaan 
0.7, which renames many of lavaan's internal functions. The package passes its 
test suite against lavaan 0.6-19, lavaan 0.6-21 (current CRAN), and the
development version of lavaan 0.7-1.
